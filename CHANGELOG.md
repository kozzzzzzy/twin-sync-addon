# Changelog

## [1.0.0] - 2024

### Added
- Initial release as Home Assistant Add-on
- Camera snapshot analysis with Gemini Vision
- User-defined "Ready State" definitions
- Memory system tracking patterns over time
- Recurring item detection
- 6 voice options (Direct, Supportive, Analytical, Minimal, Gentle Nudge, Custom)
- 7 spot type templates
- Streak tracking
- Snooze functionality
- Beautiful web UI via Ingress
- SQLite persistence
- WebSocket real-time updates
- REST API

### Architecture
- FastAPI backend
- SQLite database
- Jinja2 templates
- Vanilla JS frontend
- Docker container
